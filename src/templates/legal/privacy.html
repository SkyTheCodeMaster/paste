Privacy Policy

What type of information and why do we collect from users?

  • E-Mail Addresses: We collect E-Mail addresses for verification and password resets purposes.
  • Usernames: We collect usernames for login purposes and to represent the user in the website.
  • IP Address used to create new pastes.

What type of information and why do we collect from visitors?

Registered users included
  • User Agent: The user browser sends an information containing what type of browser, operating system to generate the session cookie along the IP Address
  • IP Address: The IP Address is used to generate the session cookie

By what means is this information collected?

We use cookies to store the following:

  • User session token: This cookie is created upon logging in and it's used to represent the user in the application.
  • Security session token: This cookie is created upon logging into the settings page and it's used to represent the user in the application.

Who has access to these information?

  • These information are only stored in the website server and in the visitor browser.

How do we update users about upcoming changes to the Privacy Policy and Terms of Service?

An announcement will be displayed at the top of the main page describing the changes and when it will be applied.

Emails may also be sent as well.

Protection

We store password as hashed data using the algorithms SHA256 and PBKDF2. Hashing is an one-way process to make the data unreadable.
